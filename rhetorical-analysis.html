Problem and Solution

  • Problem: Students in COMS 100B (Rhetorical Analysis) at Sac State University often struggle with complex writing-intensive rhetorical analysis assignments
  • Solution: A comprehensive outline document that provides detailed guidance on how to complete all the requirements for every assignment, made in close collaboration with the professor.

Skills Demonstrated

  • Technical skills: Google Docs
  • Soft skills: Creative problem-solving, attention to detail, organizational skills, collaboration, distilling information

Results

  • The project became an invaluable resource for both classmates and the professor, significantly improving understanding in the course.

Description

An outline document thoroughly explaining (via comments on the right-hand side) what to write in each section of the rhetorical analysis assignments in COMS 100B. I decided to make this document in close collaboration with the professor to help myself and my peers ensure that we were meeting all the requirements for the assignments.
